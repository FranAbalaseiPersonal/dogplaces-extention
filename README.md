# DogPlaces Chrome Extension

Shows if a Google Maps place is Dog Friendly, Terrace Only, No Dogs, or Unknown.
Fetches data from Supabase.

## Features
- Automatically reads the Google Maps URL
- Fetches dog-friendliness from Supabase
- Clean UI with collapsed/expanded cards
- Draggable widget
- Dynamic brand logo sizes

## Versioning
All releases are tagged and mapped to Chrome Web Store versions.
